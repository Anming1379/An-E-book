## 语法 - 正文

普通的文字直接输入即可, 一段过长了会自动换行, 换段落可以按两个回车

    在前面加上四个空格可以让段落有缩进, 看起来更像书本, 这一段是缩进段落

**加粗文字** :

```
**加粗文字**
```

*斜体文字* :

```
*斜体文字*
```

~~删除线~~ :

```
~~删除线~~
```

> 引用文字:

```
> 引用文字
```

单行代码:

```
单行代码可以用`括起来, 代码里的语法不会被渲染
```

多行代码

```
这里是你想要的代码
阿巴阿巴阿巴
这里的东西都不会被渲染, 前面的语法都是用这个展示的
```

```
- 吃桉
- 吃波
- 吃小梦
```

## 分段和分页

分段通常打两个回车

```
我是第一段

我是第二段
```

打出这个`<br>`符号可以强制换行<br>类似这样

```
强制换行<br>类似这样
```

分页通常会自动按照段落分, 强制分页用`---`

```
第一页的内容

---

第二页的内容
```

## 书的内容

- 首先写好你的txt或者md文件, 本书大部分语法与markdown相同
- 然后用浏览器打开 `build.html` 选中你的文件, 上传并转换成 `manifest.js`, 并把它放在根目录(就是 index.html 所在的地方)
- 最后直接用浏览器打开 `index.html` 就完成啦!

## 更换

想要更换主题, 请用编辑器打开 `index.html` 的代码, 找到第 10 行的 `<body>` 标签

- 如果什么也不写,即 `<body>` , 就是默认主题

[![默认主题](默认主题.png)

- 如果改为 `<body class="an-theme">` ,就是桉的主题

[![桉的主题, 最好看!](改主题的位置.png)

- 同理, 改为 `<body class="modern-theme">` 就是现代主题了
